# LGAudioKit_swift

## sound recorder and player

- 对AVAudioRecorder和AVAudioPlayer进行了封装，只暴露简单整洁的接口
- 内部包含了 **LGSoundPlayer** 和 **LGSoundRecorder** 两个独立类
- 录音文件是.caf格式。如果需要别的格式，自己调配。
